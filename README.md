# 🚀 Team Task Manager – Frontend

A modern **Next.js (App Router) frontend** for a full-stack Team Task Manager application.
This app allows users to manage projects, assign tasks, and track progress in real-time.

---

## 🌐 Live Demo

👉 https://task-manager-frontend-nine-puce.vercel.app/login

---

## 🧩 Features

* 🔐 JWT Authentication (Login / Signup)
* 📁 Multi-project management
* 👥 Team collaboration (members + roles)
* 📌 Task creation & assignment
* 🔄 Task status tracking (Todo → In Progress → Done)
* 📊 Dashboard with task stats
* 🛡️ Protected routes
* ⚡ Optimistic UI updates

---

## 🏗️ Tech Stack

* **Framework:** Next.js (App Router)
* **Language:** TypeScript
* **Styling:** Tailwind CSS
* **HTTP Client:** Axios
* **State:** React Hooks (useState, useEffect)

---

## ⚙️ Environment Variables

Create `.env.local`:

```
NEXT_PUBLIC_API_URL=https://your-backend-url.onrender.com/api
```
